Einrichtungs-Anleitung für macOS — einmalig, ca. 10 Minuten

Schritt 1 — Python installieren

Im Browser python.org/downloads aufrufen
→ Gelben Button klicken → .pkg-Datei öffnen → Installer bis zum Ende durchklicken.

Schritt 2 — Projektdateien herunterladen

Auf der GitHub-Seite oben rechts auf „Code" klicken
→ „Download ZIP" wählen → ZIP-Datei im Downloads-Ordner entpacken (Doppelklick).

Schritt 3 — Terminal öffnen

Tastenkombination ⌘ + Leertaste drücken
→ Terminal eintippen → Enter drücken.

Schritt 4 — In den Projektordner navigieren

Im Terminal eintippen: cd (mit Leerzeichen am Ende!)
→ Den entpackten Projektordner aus dem Finder ins Terminal-Fenster ziehen
→ Enter drücken.

Tipp: Der Ordner heißt wahrscheinlich „HasisKocht" o.ä.

Schritt 5 — Abhängigkeiten installieren (nur einmalig!)

Diesen Befehl eintippen und Enter drücken: pip3 install -r requirements.txt

Abwarten bis alles fertig geladen ist. Einmal reicht für immer.

Schritt 6 — App starten

python3 app.py Im Terminal erscheint dann: Running on http://127.0.0.1:5000

Das Terminal-Fenster offen lassen, solange du die App nutzt.

Schritt 7 — Browser öffnen & loslegen

Im Browser aufrufen: http://localhost:5000
→ YouTube-URL eingeben → Download klicken → fertig!

Beim nächsten Mal (Kurzversion)

1. Terminal öffnen (⌘ + Leertaste → Terminal)
2. cd + Projektordner reinziehen + Enter
3. python3 app.py
4. Browser → localhost:5000

Schritte 1–5 der Ersteinrichtung entfallen ab jetzt komplett.

Hinweis: Die App läuft nur lokal auf dem eigenen Mac und ist für niemanden sonst im Internet erreichbar. Kein Account, keine Cloud, keine Kosten.
